Key Difference from Private Flow

VIP Flow is simplified: Only Make, Model, and Year trigger resets. Make → Model & Trims, Model → Trims, Year → Trims. All other steps are independent.

Important Notes

  • Linear Flow: Strict step-by-step navigation
  • Minimal Resets: Only 3 reset scenarios exist:
    • Make → resets Model & Trims
    • Model → resets Trims
    • Year → resets Trims
  • No Same Value Protection: Re-selecting the same value still triggers resets
  • No Edit Mode Logic: Same behavior for create and edit
  • Trim API: Triggered after Year selection
  • Direct to Payment: After Price step
